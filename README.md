### Description

|`Folder`|`Contents`|
|:---:|---|
|[`.vscode/`](./.vscode/)|`VsCode` 에서 여러가지를 편하게 하기 위해 작성한 폴더입니다. `Clean Java class files`, `Build & Run Java` 와 같은 `tasks` 가 포함되어 있습니다.|

|[`scripts`](./scripts/)|공부한 내용을 기록한 폴더입니다. 기록 방식은 [`script_form`](./script_form/) 의 내용과 같습니다.|
