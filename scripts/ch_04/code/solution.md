문제에서 제시한 그대로 구현하면 된다.

하지만 그래도 연습을 위해 `interface` 를 이용해 추상 메서드를 선언하고, 이를 상속하는 형태로 구현해봤다.

덧붙여 `C++` 에서는 다음과 같은 문법으로, 여러개의 필드의 접근성(?) 을 쉽게 명시할 수 있었다.

```cpp
class Foo
{
    private :
        ...
    public : 
        ...
}
```

근데 `Java` 에는 이런 방법이 없다고 한다. 그래서 `default` 접근 제어자가 있나 보다.
